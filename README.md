# 📱 Calculator Pro - React Native

A clean, responsive, and functional calculator app built using **React Native** and **Expo**.

## 🚀 Features
- **Basic Math:** Addition, Subtraction, Multiplication, and Division.
- **Advanced Logic:** 
  - Percentage (%) and Sign Toggle (+/-).
  - Division by Zero handling (Displays "Error").
  - Input limit of 10 digits to prevent UI overflow.
- **UX Enhancements:** 
  - Backspace button for easy error correction.
  - Expression label showing current operation in progress.
  - Responsive "Pocket" design for all screen sizes.

## 🛠️ Tech Stack
- **Framework:** React Native (Expo)
- **Language:** TypeScript / JavaScript
- **State Management:** `useState` Hook
- **Styling:** `StyleSheet` with Flexbox layout

## 📦 How to Run
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npx expo start` to launch the app.
4. Scan the QR code using the **Expo Go** app on your mobile device.
